# 💰 Expense Tracker System

A full-stack web application that helps users efficiently manage daily expenses, track spending patterns, and maintain financial records with ease.

---

## 📌 Features

- 🔐 User Authentication (Login & Signup)  
- ➕ Add, Update, Delete Expenses  
- 📊 Track Income vs Expenses  
- 📅 Date-wise Expense Filtering  
- 📁 Expense Categorization (Food, Travel, Bills, etc.)  
- 📈 Dashboard with Summary  
- 💾 Persistent Data Storage using Database  
- 📱 Responsive UI

###2️⃣ Backend Setup (Spring Boot)

*Open project in IDE (IntelliJ / Eclipse)

*Configure application.properties:
```bash
spring.datasource.url=jdbc:mysql://expense_db
spring.datasource.username=root
spring.datasource.password=yourpassword
```
###3️⃣ Frontend Setup

*Open index.html in your browser

*OR

*Use Live Server (VS Code extension)
```
###4️⃣ Database Setup
